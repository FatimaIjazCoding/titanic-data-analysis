# Titanic Data Analysis 🚢

## 📌 Project Overview
This project analyzes the Titanic dataset to understand factors affecting passenger survival.

## 📊 Dataset
- Source: Titanic dataset
- Features include Age, Sex, Class, Fare, etc.

## 🧹 Data Cleaning
- Filled missing Age values with mean
- Removed Cabin column due to many missing values
- Filled Embarked with most frequent value

## 📈 Visualizations
- Survival count
- Survival by gender
- Survival by class
- Age distribution

## 🔍 Key Insights
- Females had higher survival rate
- First-class passengers had better survival chances
- Most passengers were in lower fare range

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
